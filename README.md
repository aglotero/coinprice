Você pode adicionar o bot no seu (FbMessanger)[https://www.messenger.com/t/CoinPriceApp] ou no (Skype)[https://join.skype.com/bot/e2840a39-7587-43c8-83dc-51d9ebbd897a] e testar uma conversa.
